A nuclear reactor simulator / game project, by Helsinki Hacklab.

Uses physical consoles with buttons and gauges for input and output.

The primary goal is to get this ready for ALT-party 2011 in Helsinki ( http://www.altparty.org/2011/ )

Design and plans in etherpad here: http://kirjoitusalusta.fi/hacklab-reactor
